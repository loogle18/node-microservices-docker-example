# node-microservices-docker-example

### Set environment variables
```
$APP_USER
$APP_PASSWORD
$USER_DB_USER_NAME
$USER_DB_PASSWORD
$USER_DB_NAME
$BOOK_DB_NAME
```

### Build and run containers
`docker-compose up`


#### Schema
```
Users:
  - first_name
  - last_name
  - age

Books:
  - title
  - text
```
